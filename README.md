# Holberton School - Low-Level Programming & Algorithm

This repository gathers all the projects and exercises dedicated to low-level programming and algorithms carried out as part of the **Holberton School** curriculum. The main goal is to understand the internal workings of computers through mastery of the C language, memory management, and data structures.

## 📝 Low-Level Curriculum Overview

The course is structured to progress from an introduction to C syntax to advanced operating system manipulation:
* **Foundations:** Syntax, variables, loops (`for`, `while`), functions, and variable scope.
* **Memory Management:** Pointers, arrays, strings, and dynamic allocation (`malloc`, `free`).
* **Data Structures:** Linked lists (single, double), stacks, queues, and binary trees.
* **System:** File handling (I/O), processes, and advanced concepts.

## 🛠️ Requirements & Standards (Style Guide)

* **Language:** C (compiled with `gcc`)
